# Configuring Windows Subsystem for Linux (WSL & WSL2)
The system resources being used can be changed by going to `WSL Settings` or by modifying your `.wslconfig` file

See: https://learn.microsoft.com/en-us/windows/wsl/wsl-config#configuration-settings-for-wslconfig

The options you want to focus on are:
- memory
- processors
